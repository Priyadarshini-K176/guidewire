# 🚀 AI-Powered Parametric Insurance for Gig Workers

## 📌 Problem Statement
India’s gig economy (Zomato, Swiggy, Amazon, Zepto, etc.) faces frequent income disruptions due to external factors like weather, pollution, and curfews. These workers lose up to **20–30% of their income** without any protection.

This project builds an **AI-powered parametric insurance platform** that provides **instant income protection** for delivery partners.

---

## 🎯 Objective
To design a smart insurance system that:
- Covers **loss of income ONLY**
- Uses **AI for risk prediction & fraud detection**
- Provides **automatic payouts**
- Operates on a **weekly pricing model**

---

## 👤 Target Persona
We focus on:
> **Food Delivery Partners (Zomato / Swiggy)**

### Scenario
- Heavy rain → delivery halted → loss of income  
- Pollution spike → unsafe conditions → reduced working hours  
- Curfew → no access to zones → zero earnings  

---

## 💡 Solution Overview

### 🧠 Key Idea
A **parametric insurance system** where:
- External triggers (rain, AQI, curfew) → detected via APIs
- Claims are **auto-triggered**
- Payout is **instant and automatic**

---

## ⚙️ System Workflow

1. User Registration (Delivery Partner)
2. Risk Profiling using AI/ML
3. Weekly Policy Generation
4. Real-time Monitoring of External Conditions
5. Trigger Detection (Weather / Pollution / Curfew)
6. Automatic Claim Initiation
7. Fraud Validation
8. Instant Payout Processing

---

## 💰 Weekly Premium Model

- Premium calculated dynamically based on:
  - Location risk (flood-prone / pollution zones)
  - Historical disruption data
  - Work patterns

### Example:
| Risk Level | Weekly Premium |
|-----------|--------------|
| Low Risk  | ₹10          |
| Medium Risk | ₹20        |
| High Risk | ₹35          |

## ⚡ Parametric Triggers

| Trigger Type | Condition | Action |
|-------------|----------|--------|
| 🌧 Rain | Rainfall > Threshold | Claim triggered |
| 🌫 Pollution | AQI > 300 | Claim triggered |
| 🚧 Curfew | Zone restricted | Claim triggered |
| 🌡 Heatwave | Temp > 45°C | Claim triggered |
